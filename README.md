# BMI 体质评估与预测系统

基于 Java Swing 的健康管理系统，采用版本化目录管理。

## 版本列表
| 版本 | 目录 | 说明 |
|------|------|------|
| v1.0 | [version1/](version1/) | 初始版本：Nimbus + 主题系统 |
| v2.0 | [version2/](version2/) | UI 全面升级：现代配色 + 圆角按钮 + 渐变图表 + 动画 |
| v3.0 | [version3/](version3/) | Aurora 设计系统：深色侧边栏 + 玻璃态登录 + 贝塞尔曲线 + 渐变按钮 |
| v4.0 | [version4/](version4/) | UI 全面重设计：Indigo/Emerald 设计令牌 + 深色侧边栏导航 + 自绘圆角按钮/表格 |

## 快速开始
1. 进入对应版本目录（如 `version1/`）
2. 按照 README.md 配置数据库
3. 双击 `run.bat` 运行

## 技术栈
- Java (JDK 8+) + Java Swing
- PostgreSQL 数据库
