# BMI 体质评估与预测系统 · v6（详细报告版）

> 在 v5 的基础上，**新增「详细健康报告」模块** —— 一键聚合全维度数据，生成可打印/导出的综合健康评估报告。
> 后续 v7 计划接入 AI 智能分析层，对报告内容进行深度解读与个性化推荐。

设计语言：**Botanical Calm（植物疗愈）**

## ✨ v6 新增特性

- 📋 **详细健康报告**：9 大板块完整健康评估
  - 01 体成分分析（BMI / 体脂 / 肌肉 / 水分 / 内脏脂肪 / 身体年龄 / 体型判定）
  - 02 代谢评估（Harris / Mifflin / 中国公式三对比 + TDEE 计算）
  - 03 健康评分五维拆解（带逐项解读）
  - 04 趋势分析（线性回归预测 + 30 天风险评估）
  - 05 目标进度追踪（完成度 + 预计天数）
  - 06 饮食营养摘要（热量 / 宏量比 / 三餐分布 / 高频食物 TOP8）
  - 07 运动记录摘要（频率 / 时长 / 消耗 / 类型分布）
  - 08 成就徽章墙
  - 09 综合健康建议（按优先级排序的个性化改善建议）
- 🖨️ **打印 / 导出 PDF**：一键调用浏览器打印对话框，可直接保存为 PDF
- 📊 **报告引擎独立**：`src/lib/report.ts` 纯函数数据聚合层，与视图完全解耦
- 🎯 **仪表盘直达入口**：首页 hero 区新增金色「生成详细报告」按钮

## 📋 报告数据流

```
用户档案 + 90天记录 + 运动 + 目标 + 饮食 + 成就
        ↓
  generateHealthReport()   ← src/lib/report.ts
        ↓
  HealthReport 对象（9 个段 + 建议）
        ↓
  ReportView.vue          ← 渲染 + 打印
```
